Automate Container Configuration

Update deployment and statefulset container images, environment variables, and replications seamlessly from your GitHub Actions or CI/CD pipelines.

Webhook Endpoint
POST https://set-img.deoops.com/api/v1/auth/setimg
bash 
curl -X POST https://set-img.deoops.com/api/v1/auth/setimg \
  -H 'Authorization: Bearer $TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "namespace": "CHANGE-ME",
    "kind": "deploy", 
    "name": "CHANGE-ME",
    "container_name": "CHANGE-ME",
    "img": "registry.com/repo/image:tag"
  }'
